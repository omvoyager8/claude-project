Problem Statement & Background

Hyperspectral Imaging (HSI) provides a rich "spectral signature" for every pixel, making it powerful for identification. However, detecting hidden or unusual objects (anomalies/targets) is computationally challenging due to the massive data volume and inherent noise. Our system provides a robust automated detection solution using advanced ML/DL techniques to achieve high accuracy and efficiency.

Proposed System & Technical Stack

Our approach integrates classical dimensionality reduction with advanced deep learning architectures. The system involves Data Preprocessing, Dimensionality Reduction (using PCA for background separation), and finally Detection using trained models.

Core Models
CNN, Autoencoder (AE), PCA+SVM

• CNNs: For spatial-spectral feature extraction
• Autoencoders: For background reconstruction and anomaly residual calculation
